# Proyecto Autolavado
Sistema de Autolavado 


### Entrar al Proyecto: cd ~/Documents/Practica/Proyecto-Autolavado
### Verificar la ls

## Intalar FastAPI y Uvicorn: pip install "fastapi[standard]"
## Instalar Pylint: pip install pylint
### Crear el entorno virtual: python -m venv venv
### Activar el Entorno: source venv/Scripts/activate


### Ejecutar el entorno: uvicorn main:app --reload 

### Ejecutar pylint: python -m pylint main.py model.py
